# Operations

## `*` (Multiplication)

The `*` operator multiplies two numbers.

```ink
var nb = 2 * 4;
```

## `/` (Division)

The `/` operator divides two numbers.

```ink
var nb = 8 / 2;
```

## `+` (Addition)

The `+` operator adds two numbers.

```ink
var nb = 2 + 2;
```

## `-` (Subtraction)

The `-` operator subtracts two numbers.

```ink
var nb = 4 - 2;
```
